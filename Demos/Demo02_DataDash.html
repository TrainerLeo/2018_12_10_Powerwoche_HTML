Mittels data- ("Data Dash") kann einem Element eine maßgeschneiderte Eigenschaft (property) hinzugefügt werden. Auf diese Weise lassen sich zusätzliche Informationen im HTML-Code hinterlegen - und auch wieder abfragen, beispielsweise mit JavaScript.

Das geht ganz leicht: JavaScript erlaubt uns, Attribute mittels getAttribute() abzufragen. Für die data- Attribute gibt es jedoch eine bessere Möglichkeit: Sie können mittels .dataset abgefragt werden.

Wirf doch einfach einen Blick in den Code, um zu sehen, wie einfach dieses Beispiel funktioniert!
